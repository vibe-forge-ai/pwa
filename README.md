# Vibe Forge PWA

This repository only hosts the standalone Vibe Forge PWA deployment.

- Source repository: `vibe-forge-ai/vibe-forge.ai`
- Build source branch: `master`
- Published branch: `gh-pages`
- Public URL: `https://vibe-forge-ai.github.io/pwa/`

The source repository triggers this repository's deployment workflow when the client module changes on `master`.
